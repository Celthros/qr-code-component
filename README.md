# Frontend Mentor - QR code component solution

This is a solution to the [QR code component challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/qr-code-component-iux_sIO_H). Frontend Mentor challenges help you improve your coding skills by building realistic projects.

## Overview

The QR code component is a responsive card built with semantic HTML5 and modern CSS, SASS/Scss featuring a prominent QR code image, a clear heading, and a concise description. It encourages users to scan the QR code to visit Frontend Mentor, supporting skill development in front-end web design. The component emphasizes accessibility, clean structure, and adaptability across devices, making it a practical example for learning and improving front-end development techniques.
